
# BOSS威胁评估系统

这是一个用于评估小队成员对于敌对人员或BOSS的威胁，并给出建议的Python程序。使用者需要输入小队成员和敌对人员/BOSS的相关信息，程序将根据各角色的属性和技能来评估威胁并提供应对方案。

# 运行环境

Python 3.x

# 安装依赖

websockets,httpx,colorlog

运行方法
在命令行中使用以下命令运行：

python 敌人威胁评估系统.py
然后按照程序提示逐步输入参数即可。

# 功能说明

该程序可以进行两种评估模式：

单个BOSS评估
多敌对人员或BOSS评估

# 单个

在单个BOSS评估模式下，用户需要输入小队成员的信息和BOSS的信息，程序将根据小队成员的属性和技能，以及BOSS的属性和弱点来评估威胁，并给出相应的应对建议。

# 多个

在多敌对人员或BOSS评估模式下，用户需要输入小队成员的信息和敌对人员/BOSS的信息，程序将根据小队成员的属性和技能，以及各敌对人员/BOSS的属性和弱点来评估威胁，并给出相应的应对建议。

程序会将评估结果保存至advice.txt文件中。

## 输入参数说明

评估模式：用户需要输入“1”或“2”，代表单个BOSS评估模式或多敌对人员或BOSS评估模式。
小队人员：用户需要输入小队成员的数量（上限为4人）。
敌对人员/BOSS人数（仅在多敌对人员或BOSS评估模式下）：用户需要输入敌对人员/BOSS的数量。
小队成员信息：用户需要输入每个小队成员的姓名、等级、生命值、攻击力、防御力、攻速、暴击率、暴击伤害和属性。
敌对人员/BOSS信息：用户需要输入每个敌对人员/BOSS的名称、等级、生命值、攻击力和弱点属性。
输出结果说明
程序会输出以下内容：

小队成员信息
敌对人员/BOSS信息
威胁评估情况：包括每个敌对人员/BOSS的威胁指数和弱点属性
建议：根据评估结果，给出相应的应对建议，包括小队成员的技能选择和执行顺序
